## 一、核心性格特质

### 1. 乙木日坐 · 外柔内刚，不服输
- **乙木** 为阴木，代表：柔顺、谦和、有韧性、曲线救国
- 坐下**酉金七杀**，七杀克日主 → 骨子里不服输，能扛压
- 木气衰微但根在乙木 → 表面谦逊柔和，内心有不可动摇的坚持
- 七杀贴身 → 逆境中反而能翻身，危机感是成长动力

**外显**：平时温和没锋芒，关键时刻爆发力强；说话直接，有时急躁

### 2. 丙午年柱 · 正印正官双修
- **丙火正印** = 学业、传承、分析、框架思维
- **午火正官**（藏干丁火食神）= 规则、秩序、责任感
- 印官双修 → 骨子里有学术气质，有自我要求，有秩序感
- 但丙午火旺，年柱为根且火旺 → 早期可能经历较大压力塑造期

**外显**：有框架思维，但不会死板；追求清晰逻辑，但能接受灰度

### 3. 癸巳月令 · 正印护身，善于学习
- **癸水正印** 贴身，但水被巳火熬干 → 印星庇护能力有限
- 巳火藏干：丙火（正印）、庚金（伤官）、戊土（劫财）
- 伤官（庚）透出 → 转化能力强，不甘平庸，有创新意识
- 劫财（戊）同藏 → 竞争意识强，物质观务实

**外显**：学习能力强，但偏向实战型学习；善于从压力中提取经验

### 4. 丙戌时柱 · 食神得势，晚年有出口
- **丙火食神** 临旺地 → 表达、创作、转化能力在晚年充分释放
- 戌藏戊土劫财、丁火食神、辛金正财
- 食神（丙）+ 食神（丁）= 双重创造引擎
- 时柱为归宿，食神居此 → 晚年有持续的表达/创作出口

**外显**：年轻时积累，中晚年爆发；最终会找到自己独特的表达方式

## 二、压力与成长模式

**七杀（酉）攻身 + 劫财（戊）争竞 = 天然压力场**

- 七杀代表：外界的挑战、竞争、压迫感
- 劫财代表：资源的争夺、关系的张力
- 火旺木焚：三火（丙×2、丁）围乙木，燥热中求生存

**应对机制**：正印（丙/癸）提供学习和规划能力，食神（丙/丁）提供转化和表达能力

**成长路径**：外压 → 学习内化 → 转化输出 → 稳定输出

## 四、行为模式

### 沟通风格
- **直接**：不绕弯子，结论先行
- **急躁**：火旺则急，有时缺乏耐心
- **坚韧**：压不垮，越压越强

### 工作风格
- 实战优先，讨厌空谈理论
- 抗压能力强，逆境中找机会
- 有竞争意识，追求领先

### 关系模式
- 外柔内刚，边界感清晰
- 不服输，但服强者
- 物质观务实，不慕虚名
